PROJECT

Title: Image caption generator using CNN and LSTM
Duration: 6 months
Guide: Dr. Ongole Gandhi
Abstract: This project leverages a Convolutional Neural Network (CNN) to extract image features and a Long Short-Term Memory (LSTM) network to generate descriptive captions. The model integrates visual and textual data, enabling coherent and contextually relevant captioning. Experimental results demonstrate the system's effectiveness in producing accurate and fluent image descriptions.
